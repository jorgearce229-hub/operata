# Operata — Trading Journal

## Setup

### 1. Supabase — Ejecutar schema
Ve a Supabase → SQL Editor → pega el contenido de `schema.sql` y ejecuta.

### 2. Instalar dependencias
```bash
npm install
```

### 3. Desarrollo local
```bash
npm run dev
```

### 4. Deploy en Vercel
1. Sube el proyecto a GitHub
2. Conecta el repo en vercel.com → New Project
3. Click Deploy — listo!

## Stack
- React + Vite
- Supabase (auth + database)
- Recharts (gráficas)
- Vercel (deploy)
